JSresolutionImg
===============

According to the resolution show photos, jQuery Plugin

#Examples
for all images
```javascript
$(document).ready(function(){
			
			$(document).resolutionImg({
									"980" : '980-src',
									"1024" : '1024-src',
									"1980" : '1980-src'
									 });
			
});

for selectors
```javascript
$(document).ready(function(){
			
			$('div#resize').resolutionImg({
									"980" : '980-src',
									"1024" : '1024-src',
									"1980" : '1980-src'
									 });
			
});
